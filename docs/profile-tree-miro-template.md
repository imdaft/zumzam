# Шаблон для визуализации в Miro

## Структура доски Miro для дерева профилей

Этот документ можно использовать как основу для создания визуальной диаграммы в Miro.

---

## Цветовая схема

- 🔵 **Синий** - Основные категории профилей
- 🟢 **Зеленый** - Подтипы
- 🟡 **Желтый** - Обязательные блоки
- 🟠 **Оранжевый** - Опциональные блоки
- 🔴 **Красный** - Важные примечания
- ⚪ **Серый** - Описание и пояснения

## 7 основных категорий (радиально от центра)

### 1. 🏢 ПЛОЩАДКА (venue)

```
┌──────────────────────────────┐
│  ПЛОЩАДКА                    │
│  🏢                          │
│                              │
│  Шаги: 6                     │
│  Подтипов: 6                 │
└──────────────────────────────┘
        │
        ├─→ Детский центр
        ├─→ Лофт / Студия
        ├─→ Кафе / Ресторан
        ├─→ Парк развлечений
        ├─→ Открытая площадка
        └─→ Другое
```

**Детальные блоки для каждого подтипа:**

#### Детский центр
- Основные параметры (вместимость, площадь, возрастные группы)
- Удобства (Wi-Fi, кухня, игровое оборудование, батуты)
- Правила (можно ли еду, парковка, залог)
- Услуги (аниматор, питание, украшение)

#### Лофт / Студия
- Основные параметры (вместимость, площадь, высота потолков)
- Удобства (Wi-Fi, кухня, освещение, звукоизоляция)
- Правила (еда, парковка, шум, время)
- Дополнительные услуги (декор, клининг)

#### Кафе / Ресторан
- Основные параметры (вместимость, отдельная комната)
- Удобства (Wi-Fi, музыка, детское меню)
- Правила (еда, торт, парковка, предоплата)
- Услуги ресторана (банкет, украшение)

## Инструкция для создания в Miro

1. **Создать доску** "Дерево создания профилей ZumZam"

2. **Добавить центральный блок** "Создание профиля"

3. **Создать 7 основных блоков** (категории) вокруг центра:
   - Использовать разные цвета
   - Добавить иконки
   - Связать стрелками с центром

4. **Для каждой категории:**
   - Добавить блоки подтипов
   - Добавить детальные блоки характеристик
   - Использовать вложенные рамки (frames)

5. **Создать общую схему шагов** (поток слева направо)

6. **Добавить сравнительную таблицу** внизу

7. **Добавить легенду** справа

8. **Использовать связи** для показа зависимостей:
   - Сплошная линия - обязательная связь
   - Пунктирная - опциональная

9. **Добавить заметки** с примечаниями и примерами

10. **Экспортировать** как изображение или PDF для документации

---

## Дополнительные элементы

- **Зоны ответственности**: Разделить доску на зоны для каждого типа профиля
- **Примеры**: Добавить примеры заполнения для каждого блока
- **Валидация**: Указать правила валидации полей
- **UX подсказки**: Добавить заметки о UX для каждого шага
