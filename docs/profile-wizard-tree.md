# Дерево разветвлений создания профилей

## Общая структура визарда

Все профили начинаются с выбора категории, затем идут специфичные шаги в зависимости от выбранного типа.

---

## 1. ПЛОЩАДКА (venue)

### Шаг 1: Выбор категории
- Выбор: **Площадка**

### Шаг 2: Основная информация (общий для всех)
- Название площадки *
- Краткое описание (30-60 символов) *
- Город *
- Адрес * (с картой)

### Шаг 3: Выбор типа площадки
Выбор подтипа определяет дальнейшие шаги:
- **kids_center** - Детский центр / Игровая комната
- **loft** - Лофт / Студия
- **cafe** - Кафе / Ресторан
- **entertainment_center** - Парк развлечений
- **outdoor** - Открытая площадка
- **other** - Другое

### Шаг 4: Визуальное оформление (опционально)
- Логотип (квадрат 500×500px)
- Обложка (16:9, 1920×1080px)

### Шаг 5: Характеристики (зависят от типа площадки)

#### 5.1. ДЕТСКИЙ ЦЕНТР (kids_center)
- **Блок: Основные параметры**
  - Вместимость (человек) *
  - Площадь (кв.м)
  - Возрастные группы (0-3, 3-6, 6-12, 12+)
  
- **Блок: Удобства и оборудование**
  - Wi-Fi для гостей
  - Кухня / Зона готовки
  - Музыкальное оборудование
  - Проектор / Экран
  - Кондиционер
  - Детская мебель (столики, стулья)
  - Игровое оборудование
  - Батут / Надувной батут
  - Горки / Лабиринты
  - Песочница
  - Спортивный инвентарь
  
- **Блок: Правила площадки**
  - Можно приносить свою еду
  - Можно приносить напитки
  - Разрешены домашние животные
  - Есть парковка
  - Доступ для колясок
  - Есть раздевалка
  - Есть туалет для детей
  - Требуется залог
  - Запрещено курение
  
- **Блок: Услуги на площадке**
  - Есть аниматор (можно заказать)
  - Организация питания
  - Организация украшения
  - Фото/видеосъемка (можно заказать)

#### 5.2. ЛОФТ / СТУДИЯ (loft)
- **Блок: Основные параметры**
  - Вместимость (человек) *
  - Площадь (кв.м)
  - Высота потолков (м)
  - Количество комнат/зон
  
- **Блок: Удобства и оборудование**
  - Wi-Fi для гостей
  - Кухня / Зона готовки (полноценная/частичная)
  - Музыкальное оборудование
  - Проектор / Экран
  - Кондиционер / Отопление
  - Освещение (естественное/студийное)
  - Декорации / Реквизит
  - Звукоизоляция
  
- **Блок: Правила площадки**
  - Можно приносить свою еду
  - Можно приносить напитки
  - Разрешены домашние животные
  - Есть парковка
  - Требуется залог
  - Ограничения по шуму
  - Ограничения по времени
  - Запрещено курение
  
- **Блок: Дополнительные услуги**
  - Организация питания
  - Организация украшения
  - Организация декораций
  - Клининг (включен/за доплату)

#### 5.3. КАФЕ / РЕСТОРАН (cafe)
- **Блок: Основные параметры**
  - Вместимость (человек) *
  - Площадь зала (кв.м)
  - Отдельная комната/зал для праздников (да/нет)
  - Вместимость отдельной комнаты
  
- **Блок: Удобства и оборудование**
  - Wi-Fi для гостей
  - Музыкальное оборудование
  - Проектор / Экран
  - Кондиционер
  - Детское меню
  - Детские стульчики
  - Детская игровая зона
  
- **Блок: Правила площадки**
  - Можно приносить свою еду (запрещено/разрешено/за доплату)
  - Можно приносить торт
  - Есть парковка
  - Доступ для колясок
  - Требуется предоплата/залог
  - Ограничения по времени
  - Запрещено курение
  
- **Блок: Услуги ресторана**
  - Организация банкета
  - Организация украшения
  - Фото/видеосъемка (можно заказать)
  - Детские программы

#### 5.4. ПАРК РАЗВЛЕЧЕНИЙ (entertainment_center)
- **Блок: Основные параметры**
  - Вместимость одновременно (человек) *
  - Общая площадь (кв.м)
  - Количество аттракционов
  - Возрастные ограничения
  
- **Блок: Удобства и оборудование**
  - Wi-Fi
  - Кафе на территории
  - Музыкальное оборудование
  - Проектор / Экран
  - Кондиционер
  - Раздевалки
  - Камеры хранения
  
- **Блок: Типы развлечений**
  - Аттракционы (список)
  - Игровые автоматы
  - Лабиринты
  - Батуты
  - Карусели
  - Другие активности
  
- **Блок: Правила площадки**
  - Можно приносить свою еду
  - Есть парковка
  - Доступ для колясок
  - Требуется залог/депозит
  - Ограничения по возрасту
  - Запрещено курение

#### 5.5. ОТКРЫТАЯ ПЛОЩАДКА (outdoor)
- **Блок: Основные параметры**
  - Вместимость (человек) *
  - Площадь (кв.м)
  - Тип покрытия (газон/песок/асфальт/другое)
  - Есть навес/беседка (да/нет)
  
- **Блок: Удобства и оборудование**
  - Электричество (есть/нет)
  - Вода (есть/нет)
  - Туалеты (есть/нет)
  - Мусорные контейнеры
  - Столы и скамейки
  - Мангал / Барбекю
  - Детская площадка на территории
  
- **Блок: Правила площадки**
  - Можно приносить свою еду
  - Можно приносить напитки
  - Разрешены домашние животные
  - Есть парковка
  - Требуется залог/депозит
  - Ограничения по времени
  - Ограничения по шуму
  - Разрешены шашлыки/барбекю
  
- **Блок: Дополнительно**
  - Защита от дождя (навес/палатки)
  - Отопление (обогреватели)
  - Освещение в вечернее время

#### 5.6. ДРУГОЕ (other)
- **Блок: Основные параметры**
  - Вместимость (человек) *
  - Площадь (кв.м)
  - Описание типа площадки (текстовое поле)
  
- **Блок: Базовые удобства** (стандартный набор)
  - Wi-Fi
  - Кухня
  - Музыкальное оборудование
  - Кондиционер
  - Парковка

### Шаг 6: Медиа (фото и видео)
- Галерея фотографий (до 6 фото)
  - Для детского центра: фото игровых зон, оборудования, интерьера
  - Для лофта: фото интерьера, зон, освещения
  - Для кафе: фото зала, детской зоны, примеров сервировки
  - Для парка: фото аттракционов, общий вид
  - Для открытой площадки: фото территории, навесов, инфраструктуры
- Видео (опционально, до 3 видео)

## 2. АНИМАТОР (animator)

### Шаг 1: Выбор категории
- Выбор: **Аниматор**

### Шаг 2: Основная информация
- Имя/Псевдоним аниматора *
- Краткое описание (30-60 символов) *
- Город (основной) *
- Работаю в городах: (множественный выбор)
- Радиус работы (в пределах города/до 50км/по договоренности)

### Шаг 3: Визуальное оформление (опционально)
- Фото профиля (круглое, 500×500px) - вместо логотипа
- Обложка (16:9, 1920×1080px) - портфолио/в работе

### Шаг 4: Специализация
- **Блок: Типы персонажей** (можно выбрать несколько)
  - Супергерои
  - Принцессы
  - Животные
  - Сказочные персонажи
  - Клоуны
  - Ростовые куклы
  - Другие персонажи (указать какие)
  
- **Блок: Возрастные группы**
  - 1-3 года
  - 3-6 лет
  - 6-10 лет
  - 10-14 лет
  - Все возрасты
  
- **Блок: Типы мероприятий**
  - Дни рождения
  - Выпускные
  - Новый год
  - Корпоративы (детские)
  - Праздники в детском саду/школе
  - Семейные праздники
  - Другие

### Шаг 5: Опыт и навыки
- **Блок: Опыт работы**
  - Сколько лет работаю аниматором
  - Количество проведенных мероприятий
  - Сертификаты/Дипломы (загрузить файлы)
  
- **Блок: Навыки**
  - Актерское мастерство
  - Фокусы
  - Мыльные пузыри
  - Аквагрим
  - Твистинг (фигуры из шариков)
  - Организация игр
  - Работа с музыкальным оборудованием
  - Работа с микрофоном
  
- **Блок: Дополнительные услуги**
  - Могу привезти реквизит
  - Могу организовать игры
  - Могу помочь с украшением
  - Работаю в паре/команде
  - Могу привести животных (каких)

### Шаг 6: Медиа
- **Блок: Фотографии костюмов** (обязательно, минимум 3)
  - Фото в разных образах
  - Фото реквизита
  - Примеры работы
  
- **Блок: Видео** (рекомендуется)
  - Видео с мероприятий (до 3 видео)
  - Видео-визитка

Определяет дальнейшие шаги:

- **science** - Научное шоу
- **magic** - Шоу фокусов
- **animals** - Шоу с животными

### Шаг 4: Визуальное оформление
- Логотип шоу
- Обложка

### Шаг 5: Характеристики (зависят от типа шоу)

#### 5.1. ШОУ МЫЛЬНЫХ ПУЗЫРЕЙ (bubble)
- **Блок: Параметры шоу**
  - Длительность (30/45/60 минут)
  - Количество участников (до 10/до 20/до 30/без ограничений)
  - Возрастная аудитория (3+/6+/все возрасты)
  
- **Блок: Программа**
  - Классическое шоу пузырей
  - Пузыри-гиганты
  - Пузыри с дымом
  - Пузыри с конфетти
  - Интерактив с детьми
  - Фотосессия в пузырях
  
- **Блок: Оборудование**
  - Собственное оборудование (да/нет)
  - Нужна ли сцена/пространство
  - Требования к помещению (размер, освещение)
  
- **Блок: Дополнительно**
  - Возможны выезды
  - Работаю в паре/соло
  - Дополнительные услуги (аквагрим, твистинг)

#### 5.2. НАУЧНОЕ ШОУ (science)
- **Блок: Параметры шоу**
  - Длительность (45/60/90 минут)
  - Количество участников (до 15/до 25/до 40)
  - Возрастная аудитория (6+/10+/все возрасты)
  
- **Блок: Программа**
  - Химические опыты
  - Физические опыты
  - Крио-опыты (жидкий азот)
  - Оптические иллюзии
  - Интерактивные эксперименты
  - Мастер-классы
  
- **Блок: Оборудование**
  - Собственное оборудование
  - Требования к помещению (вентиляция, столы)
  - Безопасность (сертификаты)
  
- **Блок: Образование ведущего**
  - Образование в области наук
  - Сертификаты безопасности

#### 5.3. ШОУ ФОКУСОВ (magic)
- **Блок: Параметры шоу**
  - Длительность (30/45/60 минут)
  - Количество участников (до 20/до 40/без ограничений)
  - Возрастная аудитория (5+/8+/все возрасты)
  
- **Блок: Тип фокусов**
  - Сценические фокусы
  - Иллюзии
  - Ментальные фокусы
  - Фокусы с участием детей
  - Интерактивные фокусы
  
- **Блок: Оборудование**
  - Собственный реквизит
  - Требования к пространству
  - Освещение

#### 5.4. ШОУ С ЖИВОТНЫМИ (animals)
- **Блок: Параметры шоу**
  - Длительность (30/45/60 минут)
  - Количество участников (до 15/до 25/до 40)
  - Возрастная аудитория (3+/5+/все возрасты)
  
- **Блок: Типы животных**
  - Птицы (какие)
  - Мелкие животные (какие)
  - Рептилии (какие)
  - Другие (указать)
  
- **Блок: Программа**
  - Дрессированные номера
  - Знакомство с животными
  - Фотосессия с животными
  - Интерактив с детьми
  
- **Блок: Безопасность и разрешения**
  - Документы на животных
  - Ветеринарные справки
  - Страховка
  - Требования к помещению

#### 5.5. КРИОШОУ (cryo)
- **Блок: Параметры шоу**
  - Длительность (30/45 минут)
  - Количество участников (до 20/до 30)
  - Возрастная аудитория (6+/10+)
  
- **Блок: Программа**
  - Опыты с жидким азотом
  - Создание облаков
  - Заморозка предметов
  - Крио-коктейли

#### 5.6. СВЕТОВОЕ ШОУ (light)
- **Блок: Параметры шоу**
  - Длительность (20/30/45 минут)

#### 5.7. ПЕННАЯ ВЕЧЕРИНКА (foam)

  - Длительность (30/45/60 минут)
  - Количество участников (до 30/до 50/без ограничений)

- **Блок: Дополнительно**
  - Нужна ли защита для мебели

### Шаг 6: Медиа

## Общие правила для всех профилей

### Обязательные поля (*)
- Всегда обязательны: название, краткое описание, город
- Для площадок: адрес
- Для выездных: радиус/города работы

### Опциональные шаги
- Визуальное оформление (логотип/обложка) - можно пропустить
- Медиа - можно пропустить или добавить позже
- Часть характеристик может быть опциональной

### Последовательность шагов
1. Выбор категории (всегда первый)
2. Основная информация (всегда второй)
3. Выбор подтипа (если есть)
4. Визуальное оформление (опционально)
5. Характеристики (специфичные для типа)
6. Медиа (фото/видео)

### Принципы стандартизации
- Четкая структура блоков для каждого типа
- Единообразие названий полей
- Понятные подсказки и примеры
- Валидация на каждом шаге
- Возможность сохранить черновик и вернуться позже

## Примечания по реализации

1. **Динамическое количество шагов**: В зависимости от категории и подтипа количество шагов может варьироваться (от 4 до 7 шагов)

2. **Условная логика**: После выбора подтипа на шаге 3, шаг 5 (Характеристики) должен показывать только релевантные блоки

3. **Валидация**: На каждом шаге проверять обязательные поля перед переходом дальше

4. **Автосохранение**: Сохранять прогресс после каждого шага в черновик

5. **Пропуск шагов**: Некоторые шаги можно пропустить, но обязательные поля должны быть заполнены перед финальным созданием профиля
